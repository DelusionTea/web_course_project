Создай многостраничный веб-сайт для театра (МХАТ имени Чехова) со следующими требованиями:

## Файловая структура:
```
project/
├── index.html
├── repertoire.html
├── about.html
├── contacts.html
├── css/
│   ├── style.css

│   ├── main.js
│   └── map.js
├── resources/
│   ├── texts/
│   │   ├── main-description.txt
│   │   ├── about-theater.txt
│   │   ├── history.txt
│   │   ├── contacts.txt
│   │   └── repertoire-items.json
│   └── images/
│       ├── logo.png

    └── theater-promo.mp4

## Обязательные требования:

### 1. HTML-страницы:

**index.html (Главная):**
- Header с логотипом и навигацией
- Hero-секция с приветствием
- Краткое описание театра (загружается из resources/texts/main-description.txt)
- Видео-презентация (HTML5 video или iframe)
- Анонсы спектаклей
- Footer с контактами

**repertoire.html (Репертуар):**
- Таблица прайс-листа с колонками: Название спектакля, Жанр, Партер, Амфитеатр, Балкон
- Данные загружаются из resources/texts/repertoire-items.json
- Фотографии спектаклей
- Описания постановок

**about.html (О театре):**
- История театра (из resources/texts/history.txt)
- Информация о руководстве с должностями (Художественный руководитель, Директор, Главный режиссёр)
- Фотогалерея
- Достижения и особенности

**contacts.html (Контакты):**
- Адрес, телефоны, email, режим работы (из resources/texts/contacts.txt)
- Интерактивная Яндекс.Карта с маркером (координаты МХАТ: 55.760837, 37.618423)
- Форма обратной связи: Имя, Email, Тема, Сообщение, кнопка Отправить

### 2. CSS требования:

**style.css:**
- CSS переменные для цветов:
  - Основной: бордовый (#8B0000)
  - Акцент: золотой (#D4AF37)
  - Фон: кремовый (#FFF8DC)
- Flexbox/Grid для макета
- Единый стиль header и footer на всех страницах
- Transitions для ссылок и кнопок
- Анимации при наведении

**responsive.css:**
- Breakpoints: 320px, 768px, 1024px, 1440px
- Бургер-меню для мобильных
- Адаптивная таблица

### 3. JavaScript требования:

**main.js:**
- Функция загрузки текстов из resources/texts/ через fetch()
- Функция загрузки JSON с репертуаром и динамическое создание таблицы
- Валидация формы (проверка email, обязательные поля)
- Плавная прокрутка к секциям
- Мобильное меню (toggle)

**map.js:**
- Инициализация Яндекс.Карты
- API подключение: https://api-maps.yandex.ru/2.1/?lang=ru_RU
- Маркер на координатах 55.760837, 37.618423
- Всплывающее окно с названием "МХАТ имени Чехова"

### 5. Дизайн-система:

**Цвета:**
- --primary-color: #8B0000 (бордовый)
- --accent-color: #D4AF37 (золотой)
- --bg-color: #FFF8DC (кремовый)
- --text-color: #2C2C2C (темно-серый)
- --white: #FFFFFF

**Типографика:**
- Заголовки: 'Playfair Display', serif
- Текст: 'Roboto', sans-serif
- h1: 48px, h2: 36px, h3: 28px, p: 16px

**Отступы:**
- section padding: 80px 0
- container max-width: 1200px
- gap в grid/flex: 30px

### 6. Обязательные функции:

- Все страницы имеют одинаковый header (logo + nav) и footer
- Навигация: Главная, Репертуар, О театре, Контакты
- Таблица репертуара генерируется динамически из JSON
- Форма валидируется перед отправкой (alert при успехе)
- Карта отображает местоположение театра с маркером
- Все тексты загружаются из файлов, а не хардкодятся в HTML
- Адаптивный дизайн для mobile, tablet, desktop
- Плавные переходы и hover-эффекты

### 7. Технические детали:

- Используй семантические HTML5 теги (header, nav, main, section, article, footer)
- Подключи reset.css для сброса стилей браузера
- Яндекс.Карты подключи через CDN с defer
- Добавь meta viewport для адаптивности
- Используй CSS Grid для общего макета, Flexbox для компонентов
- Добавь атрибуты alt ко всем изображениям
- Реализуй lazy loading для изображений

### 8. Дополнительные фичи:

- Анимация появления элементов при скролле
- Модальное окно для фотогалереи
- Индикатор загрузки при fetch запросах
- Плавное появление/скрытие мобильного меню
- Подсветка активной страницы в навигации

Создай полностью рабочий проект со всеми файлами, готовый к запуску в браузере.
